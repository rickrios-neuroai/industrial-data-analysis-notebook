# Industrial Sensor Data Analysis for Predictive Maintenance

This project demonstrates a simple industrial data analysis workflow using Python. It analyzes sensor measurements such as temperature, vibration, and current to identify abnormal operating conditions and generate basic predictive maintenance insights.

## Project Objectives

- Load and inspect industrial sensor data
- Visualize temperature and vibration trends
- Detect anomalies using threshold rules
- Generate engineering conclusions

## Project Structure

industrial-data-analysis-notebook/
- data/sample_sensor_data.csv
- notebooks/industrial_data_analysis.py
- requirements.txt
- README.md

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter / Notebook-style workflow

## Example Analysis Included

The current version analyzes:
- Temperature trend over time
- Vibration trend over time
- Simple anomaly detection based on thresholds
